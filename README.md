# health_tracker
# Інструкція для запуску проєкту Health Tracker
# Вимоги: Python 3.7+, Git (опціонально), веб-браузер
# Кроки:
# 1. Клонуйте: git clone https://github.com/sofiiavasylieva/health-tracker.git або розпакуйте ZIP.
# 2. cd шлях/до/проєкту; python -m venv venv; 
#    Windows: venv\Scripts\activate; macOS/Linux: source venv/bin/activate
#    pip install flask
# 3. Перевірте наявність app.py, templates, static.
# 4. (Опціонально) python app.py init_db
# 5. Запустіть: python app.py; відкрийте http://127.0.0.1:5001/
# 6. Зупинка: Ctrl+C
# Поради: Очистіть кеш (Ctrl+Shift+R);
